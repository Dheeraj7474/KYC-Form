# KYC Verification Form

## Overview

The KYC (Know Your Customer) Verification Form is a multi-step React application designed to collect and validate user information for KYC compliance. The application guides users through a series of steps to input their personal information, address details, and upload necessary documents. The form provides real-time validation feedback and a summary review before submission.

## Features

- Multi-step form with progress tracking.
- Validation for each input field to ensure data integrity.
- File upload functionality for identity documents.
- Summary review of all entered information before submission.
- Responsive design suitable for various screen sizes.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **React Hook Form**: A library for managing form state and validation in React applications.
- **Tailwind CSS**: A utility-first CSS framework for styling the application.
- **JavaScript ES6+**: Modern JavaScript features for cleaner code.

## Getting Started

To run this project locally, follow these steps:

### Prerequisites

Make sure you have the following installed:
- Node.js (version 14 or higher)
- npm (Node package manager)

### Installation

1. Clone the repository:

   "git clone https://github.com/Dheeraj7474/my-app.git"

2. Navigate to the project directory:

   "cd my-app"

3. Install the dependencies:

   "npm install"

### Running the Application

To start the development server, run the following command:

"npm start"

This will launch the application in your default web browser at `http://localhost:3000`.

### Building for Production

To create an optimized build of the application, run:

"npm run build"

This will generate a `build` folder containing the production-ready files.

## Usage

1. Open the application in your web browser.
2. Follow the prompts to enter your personal information, address details, and upload your identity document.
3. Review your entries on the summary page and confirm submission.
4. Upon confirmation, you will receive an alert indicating successful submission.
